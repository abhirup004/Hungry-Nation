# Hungry Nation - Food Ordering Application

Hungry Nation is a full-stack food ordering web application developed using React for the frontend and Node.js with Express for the backend. The application allows users to browse available meals, manage a cart, and place orders through a checkout workflow.

---

## Features

- Display available meals fetched from a backend API
- Add and remove items from a shopping cart
- Automatic cart total calculation
- Checkout form with input validation
- Order submission to backend
- Global state management using React Context API
- Reusable UI components such as Modal, Button, and Input
- Custom HTTP hook for API requests

---

## Technology Stack

### Frontend
- React (Vite)
- JavaScript (ES6+)
- Context API
- Custom Hooks
- CSS

### Backend
- Node.js
- Express.js
- RESTful API
- File-based JSON storage

---

## Project Structure

The project is organized into two main parts:

- `backend/` - Node.js and Express server handling meals and orders
- `src/` - React frontend including components, hooks, context store, and utilities
